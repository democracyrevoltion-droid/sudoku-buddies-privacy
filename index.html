Privacy Policy

Applications: Sudoku Buddies and Sudoku Buddies Unlocked · Developed by vinfoxgame
Last updated: July 10, 2026

vinfoxgame ("we") respects the privacy of all our users. This policy explains how the Sudoku Buddies and Sudoku Buddies Unlocked applications ("the App") handle your data.

1. Data Collected by the App

This app does not collect, gather, or transmit any personal user data to external servers whatsoever.

The following data is stored only on the user's device (via the device's Local Storage) and is never sent anywhere:

  • Game progress (levels completed)
  • Seed value of the puzzle currently being played

This data is immediately deleted if the user uninstalls the app or clears the app's data from device settings.

2. Internet Connection

The app may connect to the internet solely to load fonts from Google Fonts. This is a standard font file download, and no personal user data is sent along with this request. If there is no internet connection, the app will continue to function normally using the system's fallback fonts.

3. Advertising & Tracking

This app has no advertising, no analytics SDKs, and no user tracking of any kind.

4. In-App Purchases

This app has no in-app purchase system and does not collect any credit card or payment information.

5. Children's Data

Since the app does not collect any personal data, there is no risk to user data across any age group, including children.

6. Policy Changes

We may update this policy from time to time. Any changes will be reflected in the updated date at the top of this page.
