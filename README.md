# sudoku-buddies-privacy
นโยบายความเป็นส่วนตัว (Privacy Policy)
แอปพลิเคชัน: Sudoku Buddies และ Sudoku Buddies Unlocked · จัดทำโดย vinfoxgame
ปรับปรุงล่าสุด: 10 กรกฎาคม 2569
vinfoxgame ("เรา") เคารพความเป็นส่วนตัวของผู้ใช้งานทุกท่าน นโยบายฉบับนี้อธิบายว่าแอปพลิเคชัน Sudoku Buddies และ Sudoku Buddies Unlocked (“แอป”) จัดการข้อมูลของท่านอย่างไร

1. ข้อมูลที่แอปเก็บ
แอปนี้ไม่เก็บ ไม่รวบรวม และไม่ส่งข้อมูลส่วนบุคคลใดๆ ของผู้ใช้ไปยังเซิร์ฟเวอร์ภายนอกทั้งสิ้น

ข้อมูลต่อไปนี้จะถูกบันทึกไว้ ในเครื่องของผู้ใช้เท่านั้น (ผ่านระบบ Local Storage ของอุปกรณ์) และไม่ถูกส่งออกไปที่ใด:

ความคืบหน้าของเกม (ด่านที่เล่นผ่านแล้ว)
ค่า Seed ของปริศนาที่กำลังเล่นอยู่
ข้อมูลเหล่านี้จะถูกลบทันทีหากผู้ใช้ถอนการติดตั้งแอป หรือล้างข้อมูลแอปจากการตั้งค่าอุปกรณ์

2. การเชื่อมต่ออินเทอร์เน็ต
แอปอาจเชื่อมต่ออินเทอร์เน็ตเพื่อโหลดแบบอักษร (fonts) จาก Google Fonts เท่านั้น ซึ่งเป็นการโหลดไฟล์แบบอักษรมาตรฐาน ไม่มีการส่งข้อมูลส่วนบุคคลของผู้ใช้ไปพร้อมกับคำขอนี้ หากไม่มีอินเทอร์เน็ต แอปจะยังคงเล่นได้ตามปกติโดยใช้แบบอักษรสำรองของระบบ

3. โฆษณาและการติดตามพฤติกรรม (Advertising & Tracking)
แอปนี้ไม่มีโฆษณา ไม่มี SDK วิเคราะห์พฤติกรรมผู้ใช้ (analytics) และไม่มีการติดตามผู้ใช้ในรูปแบบใดๆ ทั้งสิ้น

4. การซื้อในแอป (In-app Purchases)
แอปนี้ไม่มีระบบการซื้อในแอป ไม่มีการเก็บข้อมูลบัตรเครดิตหรือข้อมูลการชำระเงินใดๆ

5. ข้อมูลของเด็ก
เนื่องจากแอปไม่เก็บข้อมูลส่วนบุคคลใดๆ จึงไม่มีความเสี่ยงต่อข้อมูลของผู้ใช้ทุกช่วงอายุ รวมถึงเด็ก

6. การเปลี่ยนแปลงนโยบาย
เราอาจปรับปรุงนโยบายฉบับนี้เป็นครั้งคราว หากมีการเปลี่ยนแปลงจะอัปเดตวันที่ด้านบนของหน้านี้
